# Fake News Detection 🔍

A machine learning model that classifies news articles as fake or genuine using NLP techniques.

## About
Built as part of an exploration into NLP and text classification. The model analyzes linguistic patterns and contextual features to detect misinformation.

## How it works
- Text preprocessing and feature extraction
- TF-IDF vectorization for converting text to numerical features
- Machine learning classifier (Logistic Regression / Random Forest)
- Outputs probability score — fake or genuine
